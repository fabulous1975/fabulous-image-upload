# Fabulous Image Upload

PHP ile yazılmış basit ve kullanışlı bir resim barındırma scriptidir.  
**Tamamen eğitim amaçlıdır.**

---

## 📁 Özellikler

- 📤 Yerel dosya yükleme (bilgisayardan)
- 🔗 Uzaktan URL'den resim yükleme
- 🎲 Otomatik rastgele dosya adı oluşturma
- 🖼️ HTML ve BBcode kodu üretme
- 📦 5 MB dosya boyutu sınırı
- 🛡️ Basit güvenlik kontrolleri (uzantı, boyut, tip)
- **Hayal dünyanıza kalmış güzel şeyler yapılabilir**
---

## 🚀 Kullanım

1. Dosyaları bir web sunucusuna kopyalayın (XAMPP/WAMP/LAMP)
2. `config/config.php` dosyasında `$site_url` değerini kendi site adresinize göre ayarlayın
3. `files/` klasörüne yazma izni verin (`chmod 755` veya `777`)
4. Tarayıcıdan `index.php` dosyasını açın
5. Resim yüklemeye başlayın
